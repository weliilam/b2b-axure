# B2B编辑弹框

## 📋 业务与功能

### 1.1 核心目标
> 提供B2B订单的编辑修改功能，以弹框形式展示表单，支持税号/进口商信息编辑与附件上传

### 1.2 功能清单
- **[基础信息编辑]**：8字段表单（2列4行），支持税号、EORI税号、所属国家、客户代码、进口商名称、进口商邮编、进口商城市、进口商地址
- **[协议信息编辑]**：7字段表单（2列4行），支持法人名称、职务、电话、邮箱、所在城市、信用代码、中文公司名
- **[备案附件上传]**：支持导入PDF/JPG/PNG格式附件
- **[表单校验]**：必填字段校验（税号、所属国家、客户代码）

### 1.3 交互要点
- [打开弹框]：通过动作 `openModal` 触发 → 弹框显示
- [关闭弹框]：点击取消按钮或动作 `closeModal` → 弹框关闭
- [提交表单]：点击确定按钮 → 校验表单 → 通过后触发 `onConfirm` 事件并关闭弹框
- [附件上传]：点击导入按钮 → 文件选择器 → 上传附件

## 📊 内容规划

```
B2B编辑弹框
├── 弹框标题区
│   └── 标题：修改
├── 基础信息（2列4行网格）
│   ├── 税号/TAXID（必填）
│   ├── EORI税号
│   ├── 税号/TAXID所属国家（必填，下拉选择）
│   ├── 客户代码（必填）
│   ├── 进口商名称
│   ├── 进口商邮编
│   ├── 进口商城市
│   └── 进口商地址
├── 备案附件
│   ├── 导入按钮
│   └── 提示文字
├── 协议信息（2列4行网格，全部非必填）
│   ├── 进口商法人名称（英文）
```

### 2.2 数据来源
- **数据类型**：表单输入数据
- **数据源**：用户填写 + 附件上传
- **关键字段**：
  - `taxId`: 税号/TAXID（必填）
  - `eoriTaxId`: EORI税号（选填）
  - `taxCountry`: 所属国家（必填，下拉）
  - `customerCode`: 客户代码（必填）
  - `importerName`: 进口商名称（选填）
  - `importerPostalCode`: 进口商邮编（选填）
  - `importerCity`: 进口商城市（选填）
  - `importerAddress`: 进口商地址（选填）
  - `legalNameEn`: 进口商法人名称英文（选填）
  - `legalPosition`: 法人职务（选填，默认 Director）
  - `legalPhone`: 进口商法人电话（选填）
  - `legalEmail`: 法人邮箱（选填）
  - `signCityEn`: 签署时所在城市英文（选填）
  - `creditCode`: 信用代码/所在国登记号（选填）
  - `companyNameCn`: 中文公司名离岸vat（选填）

### 2.3 内容示例
- **所属国家选项**：中国、美国、英国、德国、法国、日本
- **附件格式要求**：PDF、JPG、PNG，文件大小不超过10M
- **提示文案**："建议导入文件的格式为PDF、JPG、PNG，文件大小不超过10M"

## 🎨 布局与结构

### 3.1 整体布局
- **布局模式**：弹框居中显示，弹框宽度640px
- **容器宽度**：固定宽度640px
- **关键尺寸**：
  - 弹框头部：16px 标题，底部分割线
  - 表单区域：20px 上下内边距，24px 左右内边距
  - 表单网格：2列（1fr 1fr），行间距14px
  - 附件区域：12px 内边距，虚线边框
  - 底部按钮：12px 上下内边距，靠右排列

### 3.2 响应式适配
- 弹框为固定宽度640px，无需响应式适配

## ⚙️ Axure API 说明

### 5.1 事件列表（eventList）

| 事件名称 | Payload 类型 | 触发时机 | 说明 |
|---------|-------------|---------|------|
| `onConfirm` | `string` | 表单校验通过后点击确定 | 传递表单数据 JSON |
| `onCancel` | `string` | 点击取消按钮 | 传递空对象 JSON |

### 5.2 动作列表（actionList）

| 动作名称 | Params 类型 | 参数说明 | 功能描述 |
|---------|------------|---------|---------|
| `openModal` | `string` | 无（可传任意值） | 打开编辑弹框 |
| `closeModal` | `string` | 无（可传任意值） | 关闭编辑弹框 |

### 5.3 变量列表（varList）

| 变量名称 | 类型 | 默认值 | 说明 |
|---------|-----|-------|------|
| `modal_open` | `boolean` | `true` | 弹框是否打开 |

### 5.4 配置项列表（configList）

| 配置项名称 | 类型 | 默认值 | 说明 |
|----------|-----|-------|------|
| `title` | `string` | `修改` | 弹框顶部显示的标题文字 |

### 5.5 数据项列表（dataList）

**数据结构**：
```typescript
{
  taxId: string;              // 税号/TAXID

  importerPostalCode: string; // 进口商邮编
```
