# B2B订单列表页面规格文档

## 业务与功能

### 页面定位
B2B订单管理系统核心页面，为内部用户提供订单查询、管理和操作入口。

### 核心目标
- 快速检索和筛选订单
- 批量处理订单操作
- 查看订单详情和操作日志

### 功能清单
| 功能模块 | 具体功能 | 优先级 |
|---------|---------|--------|
| 搜索筛选 | 单号搜索、B2B单号搜索、时间范围选择、订单类型、订单状态、审核状态 | P0 |
| 批量操作 | 批量审核、撤销审核、批量修改额外服务、批量修改费用、确认费用、栏截、取消栏截、删除 | P0 |
| 单条操作 | 审核、修改、详情、日志 | P0 |
| 数据展示 | 表格列表展示、分页、行选择 | P0 |
| 数据导入 | 导入跟踪号 | P1 |
| 高级查询 | 展开/收起更多筛选条件 | P1 |

### 交互要点
- 支持多选批量操作
- 表格行悬停高亮
- 分页切换保留筛选条件
- 操作后提示

## Axure API 接口

### 5.1 事件列表（EventList）

| 事件名 | 说明 | Payload |
|--------|------|---------|
| `onSearch` | 点击查询或重置时触发，传递筛选条件 | JSON string |
| `onBatchAction` | 执行批量操作时触发，传递操作名称和选中行数 | JSON string |
| `onRowAction` | 点击行内操作时触发，传递操作名称和行序号 | JSON string |
| `onSelectionChange` | 表格行选择发生变化时触发，传递选中行 key 数组 | JSON string |

### 5.2 动作列表（ActionList）

| 动作名 | 说明 | 参数 |
|--------|------|------|
| `refresh` | 刷新表格数据，重新加载列表 | 无 |
| `resetFilters` | 重置所有筛选条件到初始状态 | 无 |
| `expandAdvanced` | 展开或收起高级查询 | `"true"` 展开 / `"false"` 收起 |
| `setPage` | 跳转到指定页码 | 页码数字字符串，如 `"2"` |

### 5.3 变量列表（VarList）

| 变量名 | 说明 |
|--------|------|
| `selected_count` | 当前选中行数量（数字） |
| `current_page` | 当前页码（数字） |
| `expanded` | 高级查询是否展开（布尔值） |

### 5.4 配置项列表（ConfigList）

| 属性ID | 类型 | 显示名 | 说明 | 默认值 |
|--------|------|--------|------|--------|
| `title` | input | 页面标题 | 页面顶部显示的标题文字 | B2B订单列表 |
| `pageSize` | inputNumber | 每页条数 | 表格每页显示的记录数 | 50 |

### 5.5 数据项列表（DataList）

| 数据名 | 说明 | 包含字段 |
|--------|------|---------|
| `orders` | 订单列表数据 | 序号、业务员、客服、订单类型、订单状态、是否首批、订单审核状态、订单来源、客户代码、运单号、客户单号、服务商单号、B2B单号 |
