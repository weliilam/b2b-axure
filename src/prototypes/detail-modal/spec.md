# B2B详情弹框

## 📋 业务与功能

### 1.1 核心目标
展示 B2B 订单的完整详情信息，支持查看基本信息、收件人信息，并可填写客户备注和配载备注。

### 1.2 功能清单
- **基本信息展示**：17 个字段（运单号、客户单号、服务商单号、B2B单号、分拣码、订单来源、订单状态、订单审核状态、创建时间、客户代码、业务员、客服员、销售产品、目的国家、清关方案、是否报关件、报关方式）
- **客户备注输入**：可编辑文本输入框
- **配载备注输入**：可编辑文本输入框
- **收件人信息展示**：11 个字段（地址类型、仓库代码、收件人姓名、省/州、城市、地址1、地址2、邮编、电话、公司名称、邮箱）
- **底部操作**：取消 / 确认按钮

### 1.3 交互要点
- 订单状态和订单审核状态根据值显示对应颜色
- 客户备注、配载备注可编辑输入
- 点击确认触发 `onConfirm` 事件，传递备注内容
- 点击取消触发 `onCancel` 事件

## 🎨 布局与结构

### 3.1 整体布局
- **布局模式**：Modal 弹框，4列网格展示
- **容器宽度**：800px
- **关键尺寸**：
  - 标签字号：12px，灰色 #909399
  - 值字号：13px，深色 #333
  - 备注输入框：等宽 2 列

### 3.2 响应式适配
不适用（弹框固定宽度）

## ⚙️ Axure API 说明

### 5.1 事件列表

| 事件名称 | Payload 类型 | 触发时机 | 说明 |
|---------|-------------|---------|------|
| `onConfirm` | `JSON string` | 点击确认时 | `{"customerRemark":"...","loadingRemark":"..."}` |
| `onCancel` | `JSON string` | 点击取消时 | 空对象 `{}` |

### 5.2 动作列表

| 动作名称 | Params 类型 | 说明 |
|---------|------------|------|
| `openModal` | `string` | 打开详情弹框 |
| `closeModal` | `string` | 关闭详情弹框 |

### 5.3 变量列表

| 变量名称 | 类型 | 默认值 | 说明 |
|---------|-----|-------|------|
| `modal_open` | `boolean` | `true` | 弹框是否打开 |

### 5.4 配置项列表

| 配置项名称 | 类型 | 默认值 | 说明 |
|----------|-----|-------|------|
| `title` | `string` | `详情` | 弹框标题 |
