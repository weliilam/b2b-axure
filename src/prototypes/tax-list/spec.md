# B2B税号管理列表

## 📋 业务与功能

### 1.1 核心目标
> 提供B2B税号信息的查询、新增、修改和删除管理功能

### 1.2 功能清单
- **[筛选查询]**：支持按客户代码、税号、创建时间、国家、状态、客户等级等条件筛选
- **[批量新增]**：点击批量新增跳转到编辑弹框页面
- **[单条操作]**：支持修改和删除单条税号记录
- **[表格展示]**：分页列表展示税号信息，支持行选择

### 1.3 交互要点
- [筛选查询]：输入条件 → 点击查询 → 刷新表格数据
- [修改操作]：点击操作列"修改" → 跳转到编辑弹框页面
- [行选择]：复选框选择 → 显示选中数量
- [分页切换]：点击页码 → 切换数据

## ⚙️ Axure API 说明

### 5.1 事件列表（eventList）

| 事件名称 | Payload 类型 | 触发时机 | 说明 |
|---------|-------------|---------|------|
| `onSearch` | `string` | 点击查询 | 传递选中行数 JSON |
| `onAdd` | `string` | 点击批量新增 | 传递空对象 |
| `onEdit` | `string` | 点击修改 | 传递当前行数据 JSON |
| `onDelete` | `string` | 点击删除 | 传递当前行数据 JSON |
| `onPageChange` | `string` | 切换分页 | 传递页码 JSON |

### 5.2 动作列表（actionList）

| 动作名称 | Params 类型 | 参数说明 | 功能描述 |
|---------|------------|---------|---------|
| `refresh` | `string` | 无 | 刷新列表数据 |
| `resetFilters` | `string` | 无 | 重置筛选条件 |

### 5.3 变量列表（varList）

| 变量名称 | 类型 | 默认值 | 说明 |
|---------|-----|-------|------|
| `selected_count` | `number` | `0` | 当前选中行数量 |
| `current_page` | `number` | `1` | 当前页码 |

### 5.4 配置项列表（configList）

| 配置项名称 | 类型 | 默认值 | 说明 |
|----------|-----|-------|------|
| `title` | `string` | `税号管理` | 页面标题 |
| `pageSize` | `number` | `20` | 每页显示条数 |
